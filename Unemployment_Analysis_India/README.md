# 📊 Unemployment Analysis During COVID-19

## Overview

This project analyzes unemployment trends in India during the COVID-19 pandemic using Python. The analysis focuses on regional unemployment, employment trends, labour participation, and the impact of COVID-19 through various statistical visualizations.

> **Note:** This project uses a publicly available demonstration dataset for educational and portfolio purposes.

---

## Objectives

- Analyze unemployment trends during COVID-19
- Compare unemployment across Indian states
- Study regional employment distribution
- Visualize labour participation rates
- Identify correlations between employment indicators

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Plotly

## Key Insights

- Several states experienced significantly higher unemployment during COVID-19.
- Employment levels varied considerably across regions.
- Labour participation showed regional differences.
- Strong relationships exist between employment and unemployment indicators.
