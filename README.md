# 🌸 Task 1: Iris Flower Classification

### Overview

Developed a machine learning model to classify Iris flowers into three species based on sepal and petal measurements.

### Key Features

* Data preprocessing and exploratory analysis
* Machine Learning model using Scikit-learn
* Interactive Streamlit web application
* Real-time prediction interface

### Technologies

* Python
* Pandas
* NumPy
* Scikit-learn
* Streamlit

---

# 🚗 Task 2: Car Price Prediction

### Overview

Built a regression model to estimate the selling price of used cars using historical vehicle data.

### Key Features

* Data Cleaning and Feature Engineering
* Exploratory Data Analysis
* Random Forest Regression
* Interactive Streamlit web application
* Model evaluation using R² Score and MAE

### Technologies

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Streamlit

# 📊 Task 3: Unemployment Analysis During COVID-19

* Interactive India Map Visualization
* Multiple Statistical Charts
